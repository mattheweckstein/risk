# Risk

A browser-based solo Risk game. Play against 1-3 AI opponents with classic Risk rules.

## Requirements

- Go 1.21+
- Node.js 18+

## Running

**Backend** (port 8080):
```bash
cd backend && go run .
```

**Frontend** (port 5173):
```bash
cd frontend && npm install && npm run dev
```

Then open http://localhost:5173.
